Création d'un site web en anglais

Informations complémentaires

Dans le cadre d'une ressource d'anglais nous avons appris à communiquer par mail, par téléphone et en direct pour l'avancement d'un projet. Pour ce faire, nous avons échanger avec d'autres élève en jouant le rôle de client mais aussi de développeur.

Nous avons donc réaliser le site web d'un restaurant asiatique fictif nommé Hanaya dont j'ai codé la page d'accueil.

Afin d'attirer les clients sur le site nous avons choisi de mettre en avant les plats servis dans le restaurant, en mettant une image imposante qui prendre les trois-quarts de l'écran. Les couleurs rouge et jaune ont aussi pour but de donner envie au client puisuqe ce sont des couleurs souvent utilisées en restauration.

Pour consulter le site cliquez sur ce lien.

Compétences du projet

Pour créer ce site, nous avons utiliser les langages HTML et CSS. Nous avons aussi utiliser du vocabulaire en anglais appris au cours des séances.
